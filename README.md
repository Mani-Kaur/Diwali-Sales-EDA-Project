# Diwali-Sales-EDA-Project

Performed Exploratory Data Analysis (EDA) on Diwali sales data to identify customer purchasing behavior and sales trends using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Female customers generated higher sales revenue
- Customers aged 26–35 were the most active buyers
- Married women showed strong purchasing behavior
- Uttar Pradesh, Maharashtra, and Karnataka had highest sales
- Food, Clothing, and Electronics were top-performing categories

## Project Features
- Data Cleaning & Preprocessing
- Handling Missing Values
- Customer Segmentation
- Data Visualization
